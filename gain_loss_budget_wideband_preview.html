Cascade Noise Figure — Friis Formula

Fsys = F1 + (F2 − 1) / G1 + (F3 − 1) / (G1·G2) + …

Where F = linear noise factor (not dB), G = linear gain. The first stage NF dominates — minimise LNA / driver NF for the best system sensitivity.
