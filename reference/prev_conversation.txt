## Interactive Components
### Create Project Modal
- Trigger: `+ CREATE PROJECT` button in topbar or overview
- Fields: PROJECT NAME (placeholder: "e.g. BLDC Motor Driver v2"), PRODUCT ID ("e.g. DPL-HW-2026-001"), DESCRIPTION (textarea), Design Type (select: RF / Digital)
- Actions: `Cancel` | `CREATE & START →`
- On submit: POST `/api/projects` → redirect to pipeline view
### Load Project
- Trigger: `LOAD PROJECT` button
- Shows list of existing projects from GET `/api/projects`
- On select: loads project into sidebar + all views
### Phase Actions
- `▶ Run P{n}` — POST `/api/projects/{id}/run` with phase param
- `APPROVE & PROCEED →` — POST `/api/projects/{id}/approve/{phase}`
- Running state: spinner + "Running…" badge
### Streaming Chat (P1)
- POST `/api/projects/{id}/chat` with message
- Display streamed response with `█` animated blink cursor
- Auto-scroll to bottom on new tokens
